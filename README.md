# 🚀 JavaScript Crash Course Notes

This repository contains my personal notes from completing a JavaScript crash course using VS Code. The goal of this project was to build a strong foundation in JavaScript by learning core concepts and practicing them with small examples.

---

## 📚 What I Learned

### 🔹 Variables & Data Types

* Learned how to store data using `let`, `const`, and (briefly) `var`
* Understood the 7 data types in JavaScript:

  * String
  * Number
  * Boolean
  * Null
  * Undefined
  * Symbol
  * Object
* Practiced string manipulation and indexing

---

### 🔹 Operators

* Arithmetic operators: `+`, `-`, `*`, `/`
* Comparison operators: `>`, `<`, `>=`, `<=`
* Equality operators:

  * `==` (loose equality)
  * `===` (strict equality - recommended)

---

### 🔹 Conditionals

* Used `if`, `else if`, and `else` statements
* Built logic based on different conditions (like age checks and user access)

---

### 🔹 Logical Operators

* `&&` (AND)
* `||` (OR)
* Learned how to combine conditions to control program flow

---

### 🔹 Truthy & Falsy Values

* Falsy values:

  * `false`, `0`, `""`, `null`, `undefined`, `NaN`
* Everything else is truthy

---

### 🔹 Ternary Operator

* Shortened `if/else` statements into one line:

```js
condition ? "true result" : "false result";
```

---

### 🔹 Loops

* `for` loops (most used)
* `while` loops
* Practiced looping through numbers and strings
* Learned DRY (Don’t Repeat Yourself)

---

### 🔹 Functions

* Created reusable blocks of code
* Learned parameters and return values
* Practiced both:

  * Regular functions
  * Arrow functions

---

### 🔹 Arrays

* Stored multiple values in a single variable
* Accessed elements using index
* Learned important array methods:

  * `.push()` (adds elements)
  * `.filter()` (returns filtered array)
  * `.map()` (transforms array)

---

### 🔹 Objects

* Stored related data using key-value pairs
* Example: user profiles
* Practiced accessing and modifying object properties

---

### 🔹 Mini Project: User Sign-Up System using console.log()

* Created a function to:

  * Accept user details
  * Create a new user object
  * Add it to a users array

---

### 🔹 DOM Manipulation

* Learned how JavaScript interacts with HTML
* Used:

  * `document.querySelector()`
  * `document.getElementById()`
* Modified:

  * Text content (`innerHTML`)
  * Styles (`.style`)

---

### 🔹 Event Handling

* Created functions that respond to user actions
* Example:

  * Changing button color
  * Toggling a sidebar menu

---
